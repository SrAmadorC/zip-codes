## About Zip Codes.

Simple API that returns all the settlements data by a valid Mexico zip code

## Requirements
PHP 8.1
Docker Compose
Composer


## Install

$ git clone https://github.com/SrAmadorC/zip-codes.git

$ cd zip-codes

$ docker-compose up -d 


## Usage

    http://localhost/api/zip-codes/{zip_code}
